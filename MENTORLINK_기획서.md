## 1. 서비스 개요

### 1.1 서비스 정의

MentorLink는 K-POP 지망생(멘티)이 자신의 퍼포먼스 영상을 업로드하면, 현업 K-POP 트레이너(멘토)로부터 전문 피드백을 받고, 우수 지망생은 K-POP 기획사와 직접 연결될 수 있는 유료 피드백 플랫폼이다.

### 1.2 핵심 가치

| 가치 | 설명 |
|------|------|
| **전문성** | HYBE, SM, JYP, YG 등 메이저 기획사 출신 트레이너가 직접 피드백 |
| **접근성** | 글로벌 K-POP 지망생 누구나 온라인으로 참여 가능 |
| **연결** | 피드백을 넘어 실제 기획사 캐스팅까지 이어지는 파이프라인 |
| **공정성** | 어드민 승인 시스템을 통한 피드백 품질 보장 |

### 1.3 서비스 흐름 요약

```
지망생(멘티)           어드민(총괄)            멘토(트레이너)         기획사
   |                      |                      |                   |
   | 1. 영상 업로드 + 결제 |                      |                   |
   |--------------------->|                      |                   |
   |                      | 2. 멘토 배정          |                   |
   |                      |--------------------->|                   |
   |                      |                      | 3. 피드백 작성      |
   |                      |<---------------------|                   |
   |                      | 4. 피드백 승인/반려    |                   |
   |                      |--------------------->|                   |
   | 5. 피드백 수신        |                      |                   |
   |<---------------------|                      |                   |
   |                      |                      |                   |
   | [캐스팅존 배지 획득 시]|                      |                   |
   |                      |                      |    6. 캐스팅존에서  |
   |                      |                      |    지망생 열람      |
   |                      |<------------------------------------------|
   |                      | 7. 오디션 요청 전달    |                   |
   |<---------------------|                      |                   |
```

## 2. 사용자 역할 정의

### 2.1 멘티 (지망생 / Applicant)

- **대상**: K-POP 데뷔를 준비하는 국내외 지망생
- **가입 조건**: 이메일 인증, 프로필 작성 필수
- **핵심 활동**: 영상 업로드, 피드백 결제, 피드백 수신, 배지 획득

### 2.2 멘토 (트레이너 / Mentor)

- **대상**: K-POP 기획사 현직/전직 트레이너
- **가입 조건**: 어드민 승인 후 멘토 권한 부여
- **핵심 활동**: 배정된 지망생 피드백 작성, 배지 부여, 포화 상태 관리

### 2.3 어드민 (총괄 관리자 / Admin)

- **대상**: 플랫폼 운영팀
- **핵심 활동**: 멘토 배정, 피드백 승인/반려, 회원 관리, 홈페이지 관리

### 2.4 기획사 (Agency / Casting Company)

- **대상**: K-POP 엔터테인먼트 기획사 담당자
- **가입 조건**: 어드민 승인 후 기획사 권한 부여
- **핵심 활동**: 캐스팅존에서 지망생 열람, 관심 지망생 등록, 오디션 요청

### 2.5 역할별 접근 권한 매트릭스

| 기능 | 멘티 | 멘토 | 어드민 | 기획사 |
|------|:----:|:----:|:-----:|:-----:|
| 회원가입/로그인 | ✅ | ✅ | ✅ | ✅ |
| 프로필 관리 | ✅ | ✅ | - | ✅ |
| 피드백 지원 (영상 업로드) | ✅ | - | - | - |
| 피드백 작성 | - | ✅ | - | - |
| 피드백 승인/반려 | - | - | ✅ | - |
| 멘토 배정 | - | - | ✅ | - |
| 캐스팅존 열람 | - | - | ✅ | ✅ |
| 오디션 요청 | - | - | - | ✅ |
| 회원 관리 | - | - | ✅ | - |
| 홈페이지 편집 | - | - | ✅ | - |
| 공지사항 관리 | - | - | ✅ | - |
| 공지사항 열람 | ✅ | ✅ | ✅ | ✅ |
| Q&A (FAQ 열람) | ✅ | ✅ | ✅ | ✅ |
| 1:1 문의 | ✅ | ✅ | - | ✅ |

## 3. User Flow

### 3.1 멘티 플로우: 영상 업로드 → 결제 → 피드백 수신

```
[홈페이지 방문]
      |
      v
[회원가입] ──────────────────────────────────────────┐
      |                                              |
      v                                              |
[이메일 인증 (인증코드 발송)]                          |
      |                                              |
      v                                              |
[기본 정보 입력]                                      |
  - 이름, 성별, 생년월일                               |
  - 키/몸무게                                         |
  - 국적, 거주지                                      |
  - 연락처                                            |
      |                                              |
      v                                              |
[프로필 완성] (선택, 나중에 가능)                       |
  - 프로필 사진 4장 (정면, 좌측45도, 우측45도, 전신)    |
  - 선호 기획사 1순위/2순위                            |
  - SNS/메신저 정보                                    |
  - 언어 능력                                         |
  - 피드백 목적 (오디션준비/실력향상/진로탐색/네트워크)  |
  - 연습 경험                                         |
      |                                              |
      v                                              |
[피드백 서비스 안내 페이지 열람] <────────────────────┘
      |
      v
[피드백 지원하기]
  - 분야 선택 (보컬/댄스/랩/퍼포먼스)
  - 영상 업로드 (필수)
  - 선호 기획사 확인
  - 동의사항 체크
      |
      v
[결제 진행] ◀── 새로 구현 필요
  - 상품 확인
  - 결제 수단 선택
  - 결제 완료
      |
      v
[지원 완료 → 상태: "대기중"]
      |
      v
[어드민이 멘토 배정 → 상태: "배정됨"]
      |
      v
[멘토가 피드백 작성 → 상태: "진행중"]
      |
      v
[어드민이 피드백 승인 → 상태: "완료"]
      |
      v
[피드백 수신]
  - 장점 / 약점 / 연습방향 / 종합의견
  - 획득 배지 확인
      |
      v
[캐스팅존 배지 획득 시]
  - 캐스팅존에 프로필 노출
  - 기획사 오디션 요청 수신 가능
```

### 3.2 어드민 플로우: 멘토 배정 및 피드백 관리

```
[어드민 대시보드]
  - 전체 통계 (회원수, 지원수, 멘토수, 세션수)
  - 최근 지원 현황
  - 시스템 알림
  - 피드백 접수 차단 토글
      |
      ├──> [지원자 관리]
      |      - 전체 지원서 목록 (필터: 상태/분야)
      |      - 개별 또는 일괄 멘토 배정
      |      - 배정 모달: 멘토 필터 (분야/소속사/상태)
      |      - 멘토 포화 상태 확인 후 배정
      |
      ├──> [멘토 관리]
      |      - 전체 멘토 목록 (필터: 상태/분야)
      |      - 멘토 통계 (배정수/세션수/평점)
      |      - 멘토 상세 (기본정보/전문분야/포화상태/계정상태)
      |
      ├──> [피드백 승인]
      |      - 대기 중인 피드백 목록
      |      - 피드백 상세 열람
      |      - 승인 또는 반려 (반려 사유 작성)
      |
      ├──> [회원 관리]
      |      - 전체 회원 목록 (필터: 상태/역할)
      |      - 고급 회원 검색 (9개 필드)
      |      - 회원 상세 (권한 토글: 멘토/캐스팅/어드민)
      |      - 피드백 이력 / 결제 이력 / 캐스팅 이력 열람
      |
      ├──> [홈페이지 편집]
      |      - 히어로 섹션 편집
      |      - 프로세스 단계 편집
      |      - 멘토 프로필 관리
      |      - 파트너사 로고 관리
      |      - 섹션 표시/숨김 & 순서 조정
      |
      └──> [공지사항 관리]
             - 공지 작성/수정/삭제
             - 공지 고정(핀) 설정
```

### 3.3 멘토 플로우: 배정된 멘티 평가

```
[멘토 대시보드]
  - 배정 학생 수
  - 총 세션 수
  - 평균 평점
  - 대기 중 피드백 수
  - 월별 차트
      |
      v
[배정된 지원자 목록]
  - 검색 (이름/이메일)
  - 우선순위 표시 (48시간 초과 = 긴급)
  - 페이지네이션 (10건/페이지)
      |
      v
[지원자 상세 → 피드백 작성]
  - 지원자 영상 시청
  - 피드백 작성:
    * 장점 (strengths)
    * 약점 (weaknesses)
    * 연습 방향 (practice direction)
    * 종합 코멘트 (overall comment)
  - 배지 부여:
    * 성실 (월 4회 이상 피드백)
    * 성장 (멘토 재량)
    * 열정 (지원 7일 이내 피드백)
    * 캐스팅존 (기획사 노출 대상)
  - 임시저장 또는 제출
      |
      v
[제출 → 어드민 승인 대기]
      |
      ├──> [승인됨] → 멘티에게 전달
      └──> [반려됨] → 반려 사유 확인 → 수정 후 재제출
```

```
[멘토 프로필 설정]
  - 기본 정보 수정 (이름/이메일/연락처)
  - 전문 정보 (분야/경력/수상)
  - 소속사 관리 (현재/이전)
  - 포화 상태 토글
    * ON: 신규 배정 차단
    * 최대 배정 수 설정
  - 프로필 이미지 변경
```

### 3.4 기획사 플로우: 캐스팅존 → 오디션 요청

```
[캐스팅존 접속] (로그인 필수)
      |
      v
[지망생 목록 열람]
  - 필터: 성별 / 포지션 (보컬/댄스/랩/연기) / 출생연도
  - 프로필 카드:
    * 사진, 이름, 성별, 출생연도
    * 포지션/분야
    * 담당 멘토명
    * 배지 획득일
    * 최근 피드백일
      |
      ├──> [관심 지망생 등록] (즐겨찾기)
      |      - 토글 방식
      |      - "관심 지망생" 탭에서 모아보기
      |
      └──> [오디션 요청]
             - 오디션 요청 버튼 클릭
             - 요청 사유 작성 (신규)
             - 어드민에게 요청 전달
             - "오디션 요청" 탭에서 현황 확인
                   |
                   v
             [어드민이 멘티에게 오디션 정보 전달]
```

## 4. 상세 기능 명세

### 4.1 인증/계정 시스템

#### F-AUTH-01: 회원가입

| 항목 | 내용 |
|------|------|
| **설명** | 이메일 기반 회원가입 |
| **필수 입력** | 이름, 성별, 이메일, 비밀번호, 생년월일, 키, 몸무게, 국적, 거주지, 연락처 |
| **이메일 인증** | 인증코드 발송 → 입력 → 확인 |
| **비밀번호 규칙** | 8자 이상, 대문자+소문자+특수문자 포함 |
| **비밀번호 확인** | 비밀번호 재입력 일치 확인 |
| **약관 동의** | 이용약관, 개인정보처리방침 동의 |

#### F-AUTH-02: 로그인/로그아웃

| 항목 | 내용 |
|------|------|
| **방식** | 이메일 + 비밀번호 |
| **UI** | 모달 팝업 (네비게이션 바에서 트리거) |
| **세션 유지** | JWT 토큰 기반 (서버 구현 시) |
| **소셜 로그인** | Phase 2에서 고려 (카카오, 구글) |

#### F-AUTH-03: 프로필 관리

| 항목 | 내용 |
|------|------|
| **프로필 사진** | 4장 업로드 (정면, 좌측45도, 우측45도, 전신) |
| **선호 기획사** | 1순위, 2순위 선택 |
| **SNS 정보** | 플랫폼 선택 + ID 입력 |
| **메신저 정보** | 플랫폼 선택 + ID 입력 |
| **언어 능력** | 언어 선택 + 수준 (상/중/하) |
| **피드백 목적** | 오디션 준비 / 실력 향상 / 진로 탐색 / 네트워크 구축 |
| **연습 경험** | 자유 서술 |

### 4.2 피드백 시스템

#### F-FB-01: 피드백 서비스 안내

| 항목 | 내용 |
|------|------|
| **페이지** | 피드백 서비스 소개 랜딩 |
| **내용** | 서비스 프로세스 설명, 배지 시스템 안내, CTA 버튼 |
| **접근** | 비로그인 사용자도 열람 가능 |

#### F-FB-02: 피드백 지원 (영상 업로드)

| 항목 | 내용 |
|------|------|
| **접근 조건** | 로그인 필수 |
| **분야 선택** | 보컬 / 댄스 / 랩 / 퍼포먼스 (단일 선택) |
| **영상 업로드** | 필수, 영상 파일 또는 영상 링크 (YouTube 등) |
| **개인 정보** | 이름, 선호 기획사 자동 채움 |
| **동의 사항** | 개인정보 제공 동의, 영상 활용 동의 |
| **접수 차단** | 어드민이 전체 접수 차단 가능 (차단 시 안내 메시지 표시) |

#### F-FB-03: 지원 현황 조회

| 항목 | 내용 |
|------|------|
| **상태 카운트** | 전체 / 대기중 / 진행중 / 완료 |
| **최근 지원 목록** | 카드 형태, 분야 배지 표시 |

```
[피드백 지원 폼 작성 완료]
         |
         v
[결제 페이지 이동]
  - 상품명: "{분야} 피드백 서비스"
  - 금액 표시
  - 결제 수단 선택
         |
         ├── 신용/체크카드
         ├── 계좌이체
         ├── 카카오페이
         └── 네이버페이
         |
         v
[PG사 결제 모듈 호출]
         |
         ├── 성공 → 지원서 생성 → "대기중" 상태
         └── 실패 → 에러 안내 → 재시도
```

| 영상 접근 제어 | 서명된 URL (Signed URL)로 영상 접근 |
| XSS 방지 | CSP 헤더, 입력값 새니타이즈 |
| CSRF 방지 | CSRF 토큰 적용 |
| SQL Injection | ORM 사용 (Prisma), 파라미터 바인딩 |
| 파일 업로드 검증 | MIME 타입, 파일 크기, 확장자 검증 |

### 11.3 개인정보 보호법 준수
