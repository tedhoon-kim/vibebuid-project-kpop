# MentorLink (멘토링크) 서비스 기획

> K-POP 지망생이 영상을 올리고, 현업 트레이너에게 피드백을 받고, 기획사와 연결되는 플랫폼

---

## 1. 서비스 한 줄 요약

**"K-POP 지망생 → 영상 업로드 → 결제 → 전문 피드백 수신 → 기획사 캐스팅 연결"**

---

## 2. 누가 쓰는 서비스인가?

이 플랫폼에는 **4가지 사용자**가 있습니다.

### 멘티 (지망생)

- K-POP 데뷔를 준비하는 국내외 지망생
- 자기 퍼포먼스 영상을 올리고, 돈을 내고, 전문가 피드백을 받는 사람

### 멘토 (트레이너)

- HYBE, SM, JYP, YG 등 메이저 기획사 출신 트레이너
- 배정받은 지망생의 영상을 보고 피드백을 써주는 사람
- 운영팀이 멘토 권한을 부여해야 활동 가능

### 어드민 (운영팀)

- 플랫폼 전체를 관리하는 총괄 관리자
- 멘토 배정, 피드백 검수, 회원 관리 등 모든 운영 업무 담당

### 기획사 (엔터테인먼트 회사)

- 우수한 지망생을 발굴하려는 기획사 담당자
- 캐스팅존에서 지망생 프로필을 보고 오디션 요청을 보내는 사람
- 운영팀이 기획사 권한을 부여해야 활동 가능

---

## 3. 전체 서비스 흐름

```
지망생                운영팀               트레이너             기획사
  |                    |                    |                   |
  | ① 영상 올리고 결제  |                    |                   |
  |------------------->|                    |                   |
  |                    | ② 적합한 멘토 배정  |                   |
  |                    |------------------->|                   |
  |                    |                    | ③ 영상 보고        |
  |                    |                    |    피드백 작성      |
  |                    |<-------------------|                   |
  |                    | ④ 피드백 내용 검수   |                   |
  |                    |    (승인 또는 반려)  |                   |
  | ⑤ 피드백 받아보기   |                    |                   |
  |<-------------------|                    |                   |
  |                    |                    |                   |
  |  [우수 지망생인 경우 캐스팅존에 프로필 공개]  |                   |
  |                    |                    |                   |
  |                    |                    |   ⑥ 캐스팅존에서   |
  |                    |                    |   지망생 발견       |
  |                    |<----------------------------------------|
  |                    | ⑦ 오디션 요청 전달   |                   |
  |<-------------------|                    |                   |
```

## 4. 멘티(지망생)가 하는 것

### 4-1. 회원가입

- 이메일 인증 후 가입
- 이름, 성별, 생년월일, 키/몸무게, 국적, 거주지, 연락처 입력
- 비밀번호: 8자 이상, 대소문자+특수문자 포함

### 4-2. 프로필 작성

- 프로필 사진 4장 (정면 / 좌측 45도 / 우측 45도 / 전신)
- 선호 기획사 1순위, 2순위
- SNS 계정, 메신저 계정
- 언어 능력 (언어 + 상/중/하)
- 피드백 목적 선택:
  - 오디션 준비
  - 실력 향상
  - 진로 탐색
  - 네트워크 구축
- 연습 경험 서술

### 4-3. 피드백 신청

1. 분야 선택 (보컬 / 댄스 / 랩 / 퍼포먼스)
2. 영상 업로드
3. 동의사항 체크 (개인정보, 영상 활용)
4. **결제**
5. 신청 완료 → "대기중" 상태

### 4-4. 피드백 받기

신청 후 진행 상태를 확인할 수 있습니다:

| 상태 | 의미 |
|------|------|
| 대기중 | 결제 완료, 멘토 배정 전 |
| 배정됨 | 멘토가 배정됨 |
| 진행중 | 멘토가 피드백 작성 중 |
| 완료 | 피드백 도착! |

피드백 내용:
- **장점** - 잘하고 있는 점
- **약점** - 보완이 필요한 점
- **연습 방향** - 앞으로 어떻게 연습하면 좋을지
- **종합 코멘트** - 전체적인 평가

### 4-5. 나의 지원 현황 페이지

- 전체/대기중/진행중/완료 건수 한눈에 보기
- 최근 지원 목록
- 획득한 배지 목록 (연도/월별)

## 5. 어드민(운영팀)이 하는 것

### 5-1. 대시보드

한눈에 보는 전체 현황:
- 총 회원 수
- 총 지원 수
- 활성 멘토 수
- 총 세션 수
- 최근 지원 목록
- 긴급 알림 (처리 안 된 건 등)

**피드백 접수 차단 기능**: 상황에 따라 전체 피드백 접수를 일시 중단할 수 있음 (차단 시 지망생에게 안내 메시지 표시)

### 5-2. 멘토 배정

지원서가 들어오면 운영팀이 적합한 멘토를 배정합니다.

- 지원서 목록에서 상태/분야별 필터링
- 개별 배정 또는 여러 건 한꺼번에 일괄 배정
- 멘토 선택 시 분야/소속사/현재 배정량 확인 가능
- 멘토가 "포화 상태"이면 배정 불가

### 5-3. 피드백 검수 (승인/반려)

멘토가 작성한 피드백을 검수합니다.

- **승인**: 피드백 내용이 적절하면 승인 → 멘티에게 자동 전달
- **반려**: 부적절하면 반려 사유 작성 → 멘토에게 돌려보내 수정 요청

> 이 과정이 있어야 피드백 품질을 보장할 수 있습니다.

### 5-4. 회원 관리

- 전체 회원 목록 조회 (상태/역할별 필터)
- 회원 검색 (이름, 이메일, 연락처, 국적, 가입일 등)
- 회원 상세에서:
  - 멘토 권한 부여/해제
  - 기획사 권한 부여/해제
  - 어드민 권한 부여/해제
  - 피드백 이력 확인
  - 결제 이력 확인

### 5-5. 멘토 관리

- 전체 멘토 목록 (활성 상태, 분야, 배정 수, 평점)
- 멘토 상세 (기본정보, 전문분야, 포화상태, 배정 용량)

### 5-6. 홈페이지 편집

홈페이지 내용을 직접 수정할 수 있습니다:
- 메인 비주얼 (배경, 타이틀, 버튼)
- 서비스 프로세스 3단계 설명
- 멘토 프로필 카드 (이름, 소속, 전문분야, 사진)
- 파트너사 로고 관리 (추가/삭제/순서 변경)
- 섹션별 보이기/숨기기, 순서 변경

### 5-7. 공지사항 관리

- 공지 작성 / 수정 / 삭제
- 카테고리: 일반 / 업데이트 / 중요 / 이벤트
- 중요 공지 상단 고정

## 6. 멘토(트레이너)가 하는 것

### 6-1. 대시보드

- 현재 배정된 학생 수
- 총 진행 세션 수
- 평균 평점
- 대기 중인 피드백 수
- 월별 활동 차트

### 6-2. 배정된 지원자 확인

- 운영팀이 배정한 지원자 목록 확인
- 이름/이메일로 검색
- **48시간 초과 시 "긴급" 표시** → 빨리 피드백 작성 필요

### 6-3. 피드백 작성

지원자의 영상을 보고 피드백을 작성합니다:

1. **장점** - 이 지망생이 잘하는 점
2. **약점** - 보완이 필요한 점
3. **연습 방향** - 구체적인 연습 가이드
4. **종합 코멘트** - 전체적인 의견

추가로 **배지**를 부여할 수 있습니다 (아래 7번 참고).

- 작성 중 **임시저장** 가능
- **제출** 하면 운영팀 검수로 넘어감

### 6-4. 피드백 이력

- 월별로 작성한 피드백 목록
- 각 피드백의 상태 확인:
  - 승인 대기 중
  - 승인됨
  - 전달 완료
  - **반려됨** → 반려 사유 확인 후 수정, 재제출 가능

### 6-5. 프로필 설정

- 기본 정보 (이름, 이메일, 연락처)
- 전문 정보 (분야, 경력, 수상 경력)
- 소속사 관리 (현재/이전)
- **포화 상태 관리**:
  - ON으로 설정하면 신규 배정이 차단됨
  - 최대 배정 수 직접 설정 (기본 10명)
  - "지금 여유 없으니 잠시 쉴게요" 기능

## 7. 캐스팅존 배지 시스템

멘토가 피드백 작성 시 우수한 지망생에게 **캐스팅존 배지**를 부여할 수 있습니다.

**캐스팅존 배지를 받은 지망생만** 기획사가 볼 수 있는 캐스팅존에 프로필이 공개됩니다.

이것이 이 플랫폼의 핵심 연결 고리입니다:

```
피드백 받기 → 멘토에게 캐스팅존 배지 획득 → 캐스팅존에 프로필 공개 → 기획사가 발견 → 오디션 연결
```

---

## 8. 기획사가 하는 것

### 8-1. 캐스팅존에서 지망생 둘러보기

캐스팅존 배지를 받은 지망생들의 프로필을 볼 수 있습니다.

**필터 조건:**
- 성별
- 포지션 (보컬 / 댄스 / 랩 / 연기)
- 출생연도 범위

**프로필 카드에 표시되는 정보:**
- 프로필 사진
- 이름, 성별, 출생연도
- 포지션/분야
- 담당 멘토 이름
- 배지 획득일
- 최근 피드백 받은 날짜

### 8-2. 관심 지망생 등록

마음에 드는 지망생을 즐겨찾기 할 수 있습니다.
- 별 아이콘 클릭으로 추가/해제
- "관심 지망생" 탭에서 모아보기

### 8-3. 오디션 요청

특정 지망생에게 오디션 기회를 주고 싶을 때:
1. "오디션 요청" 버튼 클릭
2. 요청 메시지 작성
3. 운영팀에게 전달됨
4. 운영팀이 해당 지망생에게 오디션 정보 전달
5. "오디션 요청" 탭에서 요청 현황 확인

> 기획사가 지망생에게 직접 연락하는 게 아니라, **반드시 운영팀을 거치는** 구조입니다.

## 9. 결제

> 기존 HTML에는 결제 기능이 없었으므로 새로 만들어야 합니다.

### 결제 흐름

1. 피드백 신청서 작성 완료
2. 결제 페이지로 이동
3. 상품 확인 (예: "댄스 피드백 서비스")
4. 결제 수단 선택 (카드 / 계좌이체 / 카카오페이 / 네이버페이)
5. 결제 완료 → 지원서 접수됨

### 환불 정책

- 멘토 배정 전: 전액 환불 가능
- 멘토 배정 후: 환불 불가 (또는 부분 환불 - 정책 결정 필요)

### 결제 관련 페이지

| 페이지 | 내용 |
|--------|------|
| 결제 페이지 | 상품 확인 + 결제 수단 선택 + 결제 |
| 결제 완료 페이지 | 결제 확인 + 지원 현황으로 이동 |
| 마이페이지 결제 내역 | 과거 결제 이력 확인 |

## 13. 핵심 규칙 정리

### 피드백 관련

- 결제가 완료되어야 지원서가 접수됨
- 멘토는 운영팀이 배정 (지망생이 직접 고를 수 없음)
- 멘토 피드백은 반드시 운영팀 승인 후 지망생에게 전달
- 배정 후 48시간 넘으면 멘토에게 긴급 알림
- 운영팀이 전체 피드백 접수를 일시 차단할 수 있음

### 멘토 관련

- 멘토 권한은 운영팀만 부여 가능
- 멘토가 바쁘면 "포화 상태"로 설정 → 신규 배정 차단
- 최대 배정 수 초과 시 경고

### 캐스팅 관련

- 캐스팅존 배지 없는 지망생은 기획사에게 노출 안 됨
- 기획사 오디션 요청은 반드시 운영팀 경유
- 기획사 권한은 운영팀만 부여 가능
